  # OfflinePDFViewer

**Minimalistic PDF reader. An offline in-browser setup.**
### 📌 Features
<div align="left">	
	- PDF page display
	- Zoom in/out
	- Zoom to fit width/fit page/actual size
	- Pagination
	- Page download

	Built with <a href='https://mozilla.github.io/pdf.js/' target='_blank'>PDF.js</a>
